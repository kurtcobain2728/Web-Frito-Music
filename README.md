# Music Frito Landing 🎵🍟

Bienvenido a la landing page de **Music Frito**. Este proyecto es una interfaz moderna y dinámica construida con las últimas tecnologías del ecosistema de React para ofrecer una experiencia visual atractiva que incluye animaciones fluidas y efectos de partículas.

## 🚀 Tecnologías

El proyecto está desarrollado con el siguiente stack tecnológico:

*   **[React 18](https://react.dev/)**: Librería principal para la construcción de interfaces de usuario.
*   **[Vite](https://vitejs.dev/)**: Entorno de desarrollo ultrarrápido y empaquetador moderno.
*   **[Tailwind CSS](https://tailwindcss.com/)**: Framework de CSS utility-first para un diseño ágil y responsive.
*   **[Framer Motion](https://www.framer.com/motion/)**: Librería para crear animaciones fluidas y complejas.
*   **[tsParticles](https://particles.js.org/)**: Efectos visuales de partículas interactivos para el fondo.
*   **[Lucide React](https://lucide.dev/)**: Colección de íconos hermosos y consistentes.

## 📦 Instalación y Configuración

Sigue estos pasos para correr el proyecto localmente en tu entorno de desarrollo.

### Prerrequisitos

*   Asegúrate de tener [Node.js](https://nodejs.org/) instalado (recomendado v16 o superior).

### Pasos

1.  **Clona el repositorio** (si aún no lo has hecho):
    ```bash
    git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git
    cd "Web de Music Frito"
    ```

2.  **Instala las dependencias**:
    ```bash
    npm install
    ```

3.  **Inicia el servidor de desarrollo**:
    ```bash
    npm run dev
    ```
    El servidor se iniciará (por defecto en `http://localhost:5173`). Puedes visitar la URL desde tu navegador.

## 🛠️ Comandos Disponibles

En el directorio del proyecto, puedes ejecutar:

*   `npm run dev`: Ejecuta la aplicación en modo desarrollo.
*   `npm run build`: Construye la aplicación optimizada para producción dentro de la carpeta `dist`.
*   `npm run preview`: Sirve localmente la versión de producción para probarla antes de desplegar.
*   `npm run lint`: Ejecuta ESLint para analizar estáticamente el código y encontrar posibles errores.
